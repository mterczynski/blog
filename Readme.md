## Prerequisites

You'll need to have Node.js installed to run or build the application.

## Install dependencies

```bash
npm install
```

## Running the blog locally

```bash
npm run dev
```

Open http://localhost:4321/blog/

## Building the blog

```bash
npm run build
```

The static output is generated in the `dist/` directory.
