## Prerequisites

You'll need to have [Node.js](https://nodejs.org/) installed to run or build the application.

## Install dependencies

```bash
npm install
```

## Running the blog locally

```bash
npm run dev
```

Open http://localhost:5173/blog/

## Building the blog

```bash
npm run build
```
